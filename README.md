## Manufacturing Equipment IoT and Production Management ERP System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/4728b4b9a0114b31b451b4eeeee02d9f/ghb20250918) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users (employees), maintenance staff, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Material Information Management
6. Inventory In/Out Information, Inventory List Information
7. Product Information Management
8. Customer Order Management
9. Production Plan Management
10. Equipment Information Management
11. Equipment Maintenance Information
12. Quality Control Management


**Maintenance Staff Functions:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Equipment Information
6. Equipment Maintenance Management



**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Administrator Management
4. Maintenance Staff Management
5. Employee Management
6. Material Information Management
7. Inventory In/Out Management, Inventory List Management
8. Product Information Management
9. Customer Order Management
10. Production Plan Management
11. Equipment Information Management
12. Equipment Maintenance Management
13. Quality Control Management
14. Basic Data Management (Department Management, Material Type Management, Product Type Management, Equipment Type Management)
